# Dog Breed Identification Using Transfer Learning

## 📌 Project Overview
This project implements a Dog Breed Identification system using Transfer Learning with MobileNetV2.

The model is trained using TensorFlow and Keras in Google Colab with GPU acceleration.

## 🧠 Model Used
- MobileNetV2 (Pre-trained on ImageNet)
- Transfer Learning
- Fine-tuned for image classification

## 📂 Dataset
- Cats and Dogs Filtered Dataset
- Training Images: 2000
- Validation Images: 1000
- Image Size: 224x224

## ⚙️ Technologies Used
- Python
- TensorFlow
- Keras
- Google Colab
- GitHub

## 📊 Model Performance
- Training Accuracy: ~99%
- Validation Accuracy: ~98%

## 💾 Project Files
- Dog_Breed_Identification_Using_Transfer_Learning.ipynb
- dog_breed_model.h5

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. The trained model will classify images
